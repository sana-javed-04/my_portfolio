# 🚀 Professional Frontend Portfolio Website

Welcome to my personal portfolio website! This project was built from scratch as part of my frontend development task and evaluation with **Codiora Software House**. 

The website is a clean, modern, and fully responsive single-page platform designed to showcase my skills, educational background, and recent projects.

---

## 💻 Tech Stack Used
- **HTML5:** For semantic web structure.
- **CSS3:** For modern custom styling, typography, layout grids, and visual contrast (Built without any CSS frameworks like Bootstrap/Tailwind).
- **Vanilla JavaScript:** For native DOM manipulation and smooth mobile navbar toggle controls.

---

## 🛠️ Key Features Included

- **Home Section:** Premium dark-themed hero zone with a clean developer workspace background overlay and sharp typography.
- **About Me:** Detailed bio highlighting my journey from Okara, Punjab, alongside a beautiful static **Educational Timeline** (BSCS, FSc, Matric).
- **Balanced Technical Skills:** A symmetrical grid layout evenly distributed across 3 tech core cards (4 items each) for optimal UX.
- **Featured Projects Grid:** Displays 6 of my key projects including a Car Rental Platform, Food Order Frontend, and dynamic JavaScript apps.
- **Split-Layout Contact Zone:** Features quick communication channels on one side and a responsive **Contact Message Form** on the other.
- **100% Mobile Responsive:** Custom-engineered CSS Media Queries that guarantee a flawless layout on smartphones, tablets, and laptops.

---

## 📂 File Structure
- `index.html` - The core layout containing all semantic content structures.
- `style.css` - Custom styling rules, animations, theme tokens, and media queries.
- `script.js` - Lightweight native script handling the mobile navigation drawer menu.
